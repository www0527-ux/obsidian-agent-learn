# Demo 索引

每个 demo 都应该包含：

- `README.md`：学习目标、运行方式、关键问题。
- `app/`：demo 代码。
- `tests/`：至少 1-3 个关键测试。
- `reflection.md`：做完后的复盘。

## 第一个 Demo

先做 `01-minimal-fastapi-api`。目标不是写复杂代码，而是把请求生命周期讲清楚。

## 第二个 Demo

`02-sqlalchemy-sqlite-basics`。目标是用 SQLite 跑通 SQLAlchemy 2.0 的最小链路：`engine -> model -> session -> select`。
