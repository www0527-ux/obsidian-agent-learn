# FastAPI + Agent 学习项目

这个目录是一套 Obsidian 友好的学习项目，目标是从后端基础、FastAPI、SQLAlchemy、Agent/LangGraph 一路走到最终项目：一个能把学习记录关联到 Obsidian 的智能体。

## 使用方式

1. 先看 `_meta/knowledge-map.md`，了解全局路线。
2. 每次学习只推进 1-2 个概念，配合 `notes/` 和 `exercises/`。
3. 每个概念完成后，用自己的话在 `summaries/` 写一个短总结。
4. 每个模块至少做 1-3 个独立 demo，代码放在 `projects/demos/`。
5. 最终项目放在 `projects/final-learning-agent/`。
